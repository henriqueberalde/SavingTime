# Saving Time

A tool to store time records at work

# Prerequisites
- net6.0
- Database Server (example MSSQLLocalDB)

# Installation
```
> dotnet build
```

# Usage
```
> dotnet run --project SavingTime/SavingTime.csproj
```

# Testing
```
> dotnet test
```
